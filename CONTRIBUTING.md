# Contributing

## Local setup

- Server (Rust):
  - `cd server`
  - `cargo run` (defaults to `PORT=4000`)
- Client (Vite + React):
  - `cd client`
  - `npm install`
  - `npm run dev` (dev server at `http://localhost:3000`)

Set env variables using `env.example` as a reference.

## Commit style

- Keep commits scoped and descriptive.
- Prefer small PRs with a clear purpose.

## Code of Conduct

Be respectful. See `CODE_OF_CONDUCT.md`.
